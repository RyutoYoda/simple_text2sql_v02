これらのドキュメントには[Mermaid](https://mermaid.js.org/)形式の図が含まれています。以下の方法で閲覧できます：

1. **GitHub**: GitHubは自動的にMermaid図をレンダリングします
2. **VS Code**: Markdown Preview Enhanced拡張機能などを使用
3. **オンラインビューア**: [Mermaid Live Editor](https://mermaid.live/)にコードを貼り付け

## アーキテクチャの特徴

- **レイヤードアーキテクチャ**: プレゼンテーション層、ドメイン層、インフラストラクチャ層に分離
- **依存性逆転の原則**: ドメイン層のインターフェースに依存し、実装詳細から独立
- **ファクトリパターン**: データソースコネクタの生成を統一
- **アダプタパターン**: 各データソースの差異を吸収し、統一的なインターフェースを提供
